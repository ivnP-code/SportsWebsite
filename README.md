# Sports Shopping Website

This repository contains the source code for Apex Sport, a sports shopping website.

Note: After navigating to a product page, you may need to wait up to 5 seconds for the backend to transfer the required data.

The application is deployed using Vercel for the frontend and Render for the backend.

## Running the project (locally)

Install frontend dependencies:
cd frontend && npm install

Start the backend:
cd backend
dotnet run

Start the frontend development server:
cd frontend 
npm run dev

The backend provides the data required for proper website initialization.
